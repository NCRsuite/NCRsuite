# NCR Suite — V2.5.2

## Correctif Sécurité : consignes et main courante par vacation

Cette version corrige le chargement des consignes/alertes et réorganise la main courante autour des missions planifiées.

Chaque vacation possède désormais son journal indépendant : site, agent, horaires, événements et PDF dédié. Les nouvelles entrées ne peuvent plus être créées sans mission.

Installation : voir `A_LIRE_INSTALLATION.txt` et `docs/V2.5.2_INSTALLATION.md`.
