NCR Suite V2.13.2 — Finitions Restauration

BASE OBLIGATOIRE
- V2.13.1 installée avec la migration 066.

INSTALLATION
1. Exécuter dans Supabase :
   supabase/migrations/067_restaurant_finalization_release.sql
2. Ajouter les fichiers du patch sur GitHub en conservant leurs chemins.
3. Vérifier que le build Cloudflare termine en vert.
4. Fermer complètement la PWA puis la rouvrir.

AUCUNE EDGE FUNCTION À REDÉPLOYER.

CONTRÔLES CONSEILLÉS
- Le plan de salle affiche « RESTAURATION · PLAN DE SALLE ».
- L’alerte de version disparaît après installation de la migration 067 et rechargement.
- Le menu public et la réservation restent en FR, EN, ES et IT.
- Les erreurs de chargement publiques sont affichées dans la langue sélectionnée.
- Les prix adoptent le format de la langue choisie tout en restant en euros.
