NCR SUITE V2.13.1 — PATCH TRADUCTIONS RESTAURATION

BASE REQUISE
- V2.13.0 installée avec le dernier correctif de build.

ORDRE
1. Exécuter : supabase/migrations/066_restaurant_public_translations_complete.sql
2. Ajouter tous les fichiers du patch dans GitHub en conservant les chemins.
3. Attendre le build Cloudflare vert.
4. Fermer complètement la PWA puis la rouvrir.
5. Ouvrir Restauration > Identité du menu public et cliquer sur Publier cette identité.
