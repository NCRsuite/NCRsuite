# NCR Suite V2.8.2 — Installation

1. Exécuter `supabase/migrations/047_restaurant_interactive_floor_plan.sql` dans Supabase.
2. Ajouter les autres fichiers du patch dans GitHub en conservant l’arborescence.
3. Vérifier le déploiement Cloudflare.
4. Fermer puis rouvrir la PWA pour charger `ncr-suite-shell-v2.8.2-floor-plan`.

## Tests minimum

- Passer en mode édition avec un compte propriétaire, administrateur ou manager.
- Ajouter une table, la déplacer, la redimensionner et la faire pivoter.
- Ajouter un mur et une porte.
- Recharger la page et vérifier la conservation des positions.
- Passer en mode service et modifier l’état d’une table.
- Vérifier l’affichage d’une réservation attribuée à cette table.
- Sur Essentielle, vérifier que la création d’une deuxième salle reste verrouillée.
- Sur Professionnelle, créer une deuxième salle.
