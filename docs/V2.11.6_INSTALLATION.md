# NCR Suite V2.11.6 — Installation

Cette version termine la phase 1 « Audit et fiabilité globale ».

## Ordre d’installation

1. Exécuter `supabase/migrations/059_global_observability_release_validation.sql` dans Supabase.
2. Déposer les autres fichiers du patch dans GitHub en conservant leur arborescence.
3. Vérifier le build Cloudflare.
4. Fermer complètement puis rouvrir la PWA.

## Vérifications

- Ouvrir **Administration NCR → Surveillance**.
- Vérifier que les versions frontend, base et cache affichent `2.11.6`.
- Vérifier que les objets critiques sont marqués disponibles.
- Ouvrir plusieurs pages métier puis relancer la surveillance : les sessions actives doivent apparaître.
- Tester la déconnexion réseau puis le retour en ligne.
- Vérifier que le build exécute :
  - `phase1-static-audit.mjs` ;
  - `phase1-critical-flows.mjs` ;
  - TypeScript ;
  - Vite.

## Données enregistrées par la surveillance

La surveillance conserve uniquement les informations techniques nécessaires au diagnostic : message d’erreur, page, version, navigateur, date, nombre d’occurrences et entreprise concernée. Elle ne copie pas les contenus métier complets.
