# Installation NCR Suite V2.4.12

## 1. Supabase

Aucun SQL à exécuter pour cette version. Le tableau de pilotage utilise les tables et droits déjà installés jusqu’à la migration 025.

## 2. GitHub

Décompresser le patch puis envoyer tout son contenu à la racine du dépôt V2.4.11 en conservant l’arborescence. Accepter le remplacement des fichiers existants.

Les nouveaux fichiers applicatifs sont :

- `src/features/training/qualityDashboard.ts`
- `src/features/training/qualityCsv.ts`
- `src/features/training/qualityReportPdf.ts`

## 3. Cloudflare

Attendre que le déploiement affiche `Success`.

## 4. Cache PWA

Fermer complètement NCR Suite puis rouvrir l’application. Le cache applicatif passe en V2.4.12.

## 5. Contrôles

1. Ouvrir le tableau de bord d’un espace Formation.
2. Changer la période entre 30 jours, 90 jours et 12 mois.
3. Vérifier les alertes et leurs accès directs.
4. Tester l’ouverture d’une session depuis une alerte.
5. Télécharger le rapport PDF et le fichier CSV.
6. Pour une offre Professionnelle, vérifier les indicateurs de satisfaction.

Aucun redéploiement de `process-email-queue` n’est nécessaire. Coiffure, Qonto et NCR Academy ne sont pas modifiés.
