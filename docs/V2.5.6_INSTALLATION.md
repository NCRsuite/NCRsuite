# Installation NCR Suite V2.5.6

## 1. Supabase

Exécuter `supabase/migrations/033_security_professional_supervision.sql` dans le SQL Editor et attendre **Success**.

## 2. GitHub

Déposer tout le contenu du patch à la racine de la V2.5.5 corrigée en conservant l’arborescence.

## 3. Cloudflare

Attendre la fin du build, puis fermer et rouvrir la PWA.

## Contrôles

- fonctions premium visibles et verrouillées dans les offres inférieures ;
- jusqu’à 50 agents en Professionnelle ;
- rôle Agent / Chef de poste ;
- suivi GPS rattaché à une vacation ;
- PTI, confirmations et SOS ;
- supervision et traitement des alertes.

La géolocalisation d’une PWA iPhone fonctionne lorsque l’application reste active. iOS peut suspendre le suivi lorsque Safari ou la PWA est fermé.
