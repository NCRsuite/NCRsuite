# NCR Suite V2.9.0 — Installation

## Objet

Refonte ergonomique des plannings Coiffure, Formation, Nettoyage et Restauration, sans modifier le planning Sécurité ni les données existantes.

## Installation

1. Aucun SQL Supabase à exécuter.
2. Remplacer ou ajouter les fichiers du patch dans GitHub en conservant leur arborescence.
3. Vérifier le déploiement Cloudflare Pages.
4. Fermer complètement la PWA puis la rouvrir.

Cache attendu : `ncr-suite-shell-v2.9.0-premium-planning`.

## Contrôles

- Coiffure : grille par collaborateur, filtres et vue mobile.
- Formation : calendrier mensuel, sélection d’un jour et vue liste.
- Nettoyage : semaine, mois, jour et création d’une intervention.
- Restauration : semaine, jour, filtres par poste et employé.
- Sécurité : vérifier que le planning QG existant n’a pas changé.
