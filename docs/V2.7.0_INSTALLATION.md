# NCR Suite V2.7.0 — Installation du pack Nettoyage

## 1. Supabase

Exécuter dans l’éditeur SQL Supabase :

`supabase/migrations/043_cleaning_pack_core.sql`

Ce script active le domaine Nettoyage, crée les tables, les règles RLS, les fonctions de pointage, la préfacturation, les invitations agents et le bucket `cleaning-photos`.

## 2. GitHub

Remplacer ou ajouter uniquement les fichiers présents dans le patch V2.7.0 en conservant l’arborescence.

## 3. Cloudflare

Vérifier qu’un nouveau déploiement apparaît sur le commit GitHub puis attendre le statut réussi.

## 4. Cache PWA

Fermer complètement NCR Suite puis la rouvrir. Si l’ancien menu reste visible, supprimer les données du site ou réinstaller la PWA.

## 5. Tests recommandés

1. Créer une entreprise Nettoyage en offre Découverte.
2. Créer un client, un site et un agent.
3. Planifier une intervention et générer une préfacture.
4. Vérifier que l’espace agent, les rapports, anomalies, qualité et stocks apparaissent verrouillés.
5. Passer en Essentielle, inviter un agent et tester arrivée, photo avant, rapport, photo après et départ.
6. Passer en Professionnelle et tester Chef d’équipe, anomalie, contrôle qualité et seuil de stock.
