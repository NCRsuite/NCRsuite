# NCR Suite V2.8.0 — Installation du Pack Restauration

Cette version doit être installée sur la base stable V2.7.1.

## 1. Supabase

Exécuter entièrement, dans l’éditeur SQL Supabase :

```text
supabase/migrations/045_restaurant_pack_core.sql
```

La migration :

- active le domaine Restauration ;
- ajoute les employés et leur planning ;
- ajoute la carte, les catégories, les allergènes et les coûts matière séparés ;
- ajoute les fournisseurs, les stocks et les pertes ;
- ajoute les tables et les réservations ;
- ajoute les températures et les checklists ;
- étend les invitations d’équipe aux employés Restauration ;
- installe les fonctions publiques du menu QR et de réservation ;
- applique les règles RLS et les contrôles d’offre côté serveur.

Ne pas exécuter les anciens SQL une seconde fois.

## 2. GitHub

Copier tous les autres fichiers du patch dans le dépôt GitHub en conservant exactement leur arborescence.

Les fichiers déjà présents doivent être remplacés. Les nouveaux écrans Restauration doivent être ajoutés.

## 3. Cloudflare

Après le commit sur la branche de production, vérifier qu’un nouveau déploiement Cloudflare est créé et que le build se termine avec succès.

Commande de build attendue :

```text
npm run build
```

## 4. Cache PWA

Fermer complètement NCR Suite puis la rouvrir. Le cache attendu est :

```text
ncr-suite-shell-v2.8.0-restaurant
```

## 5. Contrôles après déploiement

Tester successivement :

1. création d’un espace Restauration ;
2. création d’un employé et d’un service ;
3. création d’une catégorie et d’un plat avec allergènes ;
4. ajout d’un fournisseur et d’un produit en stock ;
5. création d’une réservation interne ;
6. offre Essentielle : invitation d’un employé, table, menu QR, réservation publique, température et checklist ;
7. offre Professionnelle : rôle Manager, coût matière, marge, valeur de stock et perte ;
8. vérification des cadenas avec l’offre Découverte.

Les traductions EN, ES et IT sont modifiables dans la fiche du plat. Lorsqu’une traduction est vide, le menu public reprend la description française.
