# Installation NCR Suite V2.4.10

## 1. Supabase

Aucun SQL à exécuter.

## 2. GitHub

Décompresser le patch à la racine du dépôt V2.4.9 et remplacer uniquement les fichiers fournis.

## 3. Cloudflare

Attendre la fin du build automatique et le statut `Success`.

## 4. PWA

Fermer complètement NCR Suite puis la rouvrir afin de charger le cache V2.4.10.

## 5. Contrôle

Dans le compte super administrateur, sélectionner une entreprise Formation puis changer sa formule. Vérifier les tarifs catalogue suivants : Découverte 39,90 €, Essentielle 69,90 €, Professionnelle 99,90 €, Métier 149,90 € HT/mois.
