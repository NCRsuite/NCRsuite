# NCR Suite V2.8.1 — Installation du correctif Restauration

Ce correctif doit être installé sur une base V2.8.0 où la migration `045_restaurant_pack_core.sql` a déjà été exécutée.

## 1. Supabase SQL

Exécuter intégralement, dans l’éditeur SQL Supabase :

`supabase/migrations/046_restaurant_stabilization_translation.sql`

Cette migration répare les droits d’offres et les modules existants, ajoute les colonnes de traduction et corrige les checklists. Elle ne supprime aucune donnée Restauration existante.

## 2. Déployer la fonction de traduction

Depuis un terminal connecté au projet Supabase :

```bash
supabase functions deploy translate-restaurant-menu
```

Aucune clé n’est obligatoire : sans configuration supplémentaire, la fonction utilise son moteur automatique de secours.

Pour utiliser DeepL, recommandé pour une meilleure régularité :

```bash
supabase secrets set DEEPL_API_KEY="votre-cle-deepl"
supabase functions deploy translate-restaurant-menu
```

Option facultative pour identifier les requêtes du moteur de secours :

```bash
supabase secrets set TRANSLATION_CONTACT_EMAIL="votre-adresse@email.fr"
```

Ne jamais placer une clé DeepL dans le code React, dans `.env` exposé à Vite ou dans GitHub.

## 3. GitHub et Cloudflare

Ajouter ou remplacer les autres fichiers du patch en conservant exactement l’arborescence, puis vérifier qu’un nouveau déploiement Cloudflare est créé et termine avec succès.

## 4. Cache PWA

Fermer complètement NCR Suite puis la rouvrir. Le cache attendu est :

`ncr-suite-shell-v2.8.1-restaurant-stable`

## 5. Contrôles rapides

1. En offre Découverte : créer un employé, un service, une catégorie, un plat, un fournisseur, un stock et une réservation interne.
2. En offre Essentielle : créer une catégorie puis un plat en français ; les champs EN, ES et IT doivent se remplir automatiquement.
3. Ouvrir le menu QR public et changer de langue ; le nom du plat, sa catégorie, sa description et ses allergènes doivent changer.
4. Créer une table, un relevé de température et une checklist, puis valider la checklist.
5. En offre Professionnelle : enregistrer un coût matière et une perte.

Si la carte affiche « fonction de traduction indisponible », la fonction Supabase n’a pas été déployée sur le bon projet ou le navigateur utilise encore l’ancien cache.
