# NCR Suite V2.11.1 — Assistance sécurisée et conversation support

## Ordre d'installation

1. Exécuter `supabase/migrations/055_secure_support_assistance_chat.sql` dans Supabase.
2. Déposer les autres fichiers du patch dans GitHub en conservant l'arborescence.
3. Vérifier le build Cloudflare Pages.
4. Fermer puis rouvrir la PWA afin de charger le cache `ncr-suite-shell-v2.11.1-secure-support`.

## Parcours de test

1. Une entreprise ouvre un ticket dans **Assistance NCR**.
2. L'administrateur NCR répond depuis **Administration NCR → Support**.
3. L'administrateur demande une prise en main de 15, 30 ou 60 minutes.
4. Le propriétaire ou un administrateur de l'entreprise autorise la demande.
5. L'administrateur NCR démarre la session puis entre dans l'entreprise.
6. Un bandeau permanent indique la session active et permet de la terminer immédiatement.
7. Le client peut révoquer l'accès depuis le ticket.

## Restrictions

Les pages sensibles liées à l'abonnement, aux accès équipe, à la personnalisation, à l'offre Métier et aux paramètres sont retirées pendant la prise en main. La session expire automatiquement et reste reliée au ticket ainsi qu'au journal d'audit.
