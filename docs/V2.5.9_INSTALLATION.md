## 1 — Supabase SQL

## 2 — Edge Function Supabase

Déployer le dossier `supabase/functions/send-security-document` sous le nom exact `send-security-document`.

Avec la CLI Supabase :

```bash
supabase functions deploy send-security-document --project-ref VOTRE_PROJECT_REF
```

La fonction réutilise les secrets déjà nécessaires à Brevo :

- `BREVO_API_KEY`
- `BREVO_SENDER_EMAIL`
- `BREVO_SENDER_NAME`

Ne pas modifier ou redéployer `process-email-queue`.
