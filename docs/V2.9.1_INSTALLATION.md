# NCR Suite V2.9.1 — Réservations Restauration Premium

## Installation

1. Aucun SQL Supabase n’est nécessaire.
2. Remplacer les fichiers du patch dans GitHub en conservant leur arborescence.
3. Vérifier que Cloudflare Pages déploie le commit avec succès.
4. Fermer complètement la PWA puis la rouvrir.

## Cache PWA

`ncr-suite-shell-v2.9.1-restaurant-bookings`

## Contrôles rapides

- Ouvrir Restauration > Réservations.
- Vérifier la navigation entre les jours de la semaine.
- Tester les filtres Matin, Midi, Soir et la recherche.
- Créer une réservation avec les boutons de couverts et une carte de table.
- Vérifier qu’une table occupée sur le créneau reste indisponible.
- Tester les actions Confirmer, Installer et Terminer.
- Vérifier l’affichage sur téléphone.
