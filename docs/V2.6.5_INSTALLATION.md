# Installation V2.6.5 FINAL

Cette archive remplace le premier patch V2.6.5. Elle doit être appliquée directement sur la V2.6.4.

## 1. Supabase SQL
Exécuter entièrement `supabase/migrations/042_security_delete_planned_shift.sql`, puis attendre `Success`.

La fonction autorise la suppression définitive d’une mission uniquement lorsqu’elle est future, encore planifiée et totalement vierge de données terrain ou de facturation.

## 2. Edge Function obligatoire
Déployer une nouvelle fonction nommée exactement `send-security-document-v2` avec le contenu de `supabase/functions/send-security-document-v2/index.ts`. Laisser la vérification JWT activée. Les secrets Brevo existants sont réutilisés.

## 3. GitHub
Envoyer tout le contenu du patch à la racine de la V2.6.4, en conservant l’arborescence et en acceptant les remplacements.

## 4. Cloudflare
Attendre le statut `Success`, puis fermer et rouvrir complètement la PWA.
