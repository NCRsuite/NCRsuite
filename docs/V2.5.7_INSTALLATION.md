# Installation NCR Suite V2.5.7

Base requise : **V2.5.6**.

## 1. Supabase

Exécuter entièrement :

`supabase/migrations/034_security_web_shift_presence.sql`

Attendre le message **Success**.

## 2. GitHub

Déposer ensuite tous les fichiers du patch à la racine du dépôt, en conservant leur arborescence et en remplaçant les fichiers existants.

## 3. Cloudflare

Attendre le statut **Success** du déploiement.

## 4. PWA

Fermer complètement NCR Suite puis la rouvrir afin de charger le cache V2.5.7.

## Contrôles

1. Ouvrir un compte Agent ou Chef de poste.
2. Choisir une vacation en cours.
3. Démarrer le mode vacation.
4. Vérifier les indicateurs Réseau, Application visible, GPS et Écran maintenu.
5. Couper puis rétablir le réseau et vérifier la synchronisation des positions en attente.
6. Ouvrir la supervision avec un compte responsable.
7. Vérifier le statut de connexion de l'agent et l'heure du dernier battement.
8. Tester le compte à rebours PTI et le SOS avec une connexion active.

## Limite PWA iPhone

Le maintien d'écran et le battement améliorent la fiabilité tant que l'application reste active. iOS peut néanmoins suspendre une PWA placée en arrière-plan ou complètement fermée. La supervision affiche désormais clairement cette interruption au lieu de laisser croire que le GPS continue.
