# NCR Suite V2.5.2 — Installation

1. Exécuter `supabase/migrations/029_security_logbook_by_shift.sql` dans Supabase SQL Editor.
2. Attendre le message `Success`.
3. Envoyer le contenu du patch à la racine du dépôt GitHub V2.5.1.
4. Attendre la réussite du build Cloudflare.
5. Fermer puis rouvrir la PWA pour charger le cache V2.5.2.
6. Tester les consignes avec un compte agent, puis une main courante liée à une mission et son PDF.
