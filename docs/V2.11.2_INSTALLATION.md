# NCR Suite V2.11.2 — Finalisation de la phase 2

Cette version termine le chantier **Administration NCR, onboarding et exploitation SaaS**.

## Installation

1. Exécuter `supabase/migrations/056_saas_phase2_completion.sql` dans le SQL Editor Supabase.
2. Ajouter les autres fichiers du patch au dépôt GitHub en conservant leur arborescence.
3. Vérifier le déploiement Cloudflare Pages.
4. Fermer complètement la PWA puis la rouvrir.

Cache attendu : `ncr-suite-shell-v2.11.2-phase2-complete`.

## Fonctions ajoutées

- Centre de démarrage universel accessible aux responsables de chaque entreprise.
- Checklist calculée à partir des vraies données de chaque métier.
- Import guidé par fichiers CSV exportés depuis Excel, avec modèles, aperçu, détection des doublons et rapport d’erreurs.
- Historique audité des imports.
- Diagnostic central par entreprise dans Administration NCR.
- Contrôle des abonnements, propriétaires, quotas, onboarding, e-mails, notifications push, support, activité et stockage.
- Export JSON d’un instantané de support, sans secrets bancaires ni identifiants privés des prestataires.

## Test rapide

- Ouvrir une entreprise avec un rôle propriétaire ou administrateur.
- Aller dans `Démarrage`.
- Télécharger un modèle CSV puis importer une ligne de test.
- Valider le test de mise en service.
- Ouvrir `/administration-ncr`, puis la rubrique `Diagnostic`.
- Sélectionner l’entreprise et exporter son instantané de support.
