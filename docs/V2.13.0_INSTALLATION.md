# Installation NCR Suite V2.13.0 — Restauration premium

## Base requise

La V2.12.3 stable, incluant le correctif de cadrage Fidélité PC et de barre mobile.

## 1. Migration Supabase

Dans **SQL Editor**, exécuter :

```text
supabase/migrations/065_restaurant_public_menu_premium.sql
```

La migration ajoute les réglages du menu public, sécurise les droits, enrichit la fonction publique du menu et aligne l’état de release sur la V2.13.0.

## 2. Dépôt GitHub

Copier les fichiers du patch en conservant leur arborescence. Aucun fichier absent du ZIP ne doit être supprimé.

## 3. Déploiement Cloudflare

Le build doit terminer en vert et publier la version `2.13.0` avec le cache :

```text
ncr-suite-shell-v2.13.0-restaurant-premium
```

## 4. Validation fonctionnelle

Tester la personnalisation de l’enseigne, les six thèmes, les deux dispositions, l’import de photos, le menu public multilingue, le QR code et la réservation publique.

## Compatibilité

La migration conserve l’autorisation spéciale permettant aux entreprises Sécurité de toutes les offres d’importer leur logo documentaire. NCR Academy n’est pas concerné.
