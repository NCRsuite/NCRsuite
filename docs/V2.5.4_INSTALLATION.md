# NCR Suite V2.5.4 — Installation

1. Exécuter `supabase/migrations/031_security_patrol_billing_access_fix.sql` dans Supabase SQL Editor.
2. Attendre le message `Success`.
3. Déposer le contenu du patch à la racine du dépôt GitHub V2.5.3.
4. Attendre la réussite du build Cloudflare.
5. Fermer complètement la PWA puis la rouvrir.
6. Tester la validation d’un QR, l’actualisation de la facturation et le PDF.

La migration corrige la validation des scans et affiche tous les sites actifs du client dans l’aperçu de facturation. Seuls les sites possédant des heures programmées sont facturés.
