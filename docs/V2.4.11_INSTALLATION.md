# Installation NCR Suite V2.4.11

## 1. Supabase — obligatoire en premier

Dans **Supabase → SQL Editor**, exécuter entièrement :

```text
supabase/migrations/025_training_session_closure.sql
```

Attendre le message de réussite avant de continuer.

## 2. GitHub

Décompresser le patch à la racine du dépôt V2.4.10 et envoyer tous les fichiers en conservant l’arborescence. Accepter le remplacement des fichiers existants.

## 3. Cloudflare

Attendre que le déploiement affiche **Success**. Aucune nouvelle dépendance npm n’est ajoutée.

## 4. Cache PWA

Fermer complètement NCR Suite, puis rouvrir l’application.

## 5. Tests recommandés

1. Super administrateur : vérifier le regroupement des entreprises par domaine.
2. Compte possédant plusieurs domaines : ouvrir **Mon abonnement** et vérifier toutes les formules.
3. Formation : vérifier les onglets Planifiées, En cours, Clôturées et Annulées.
4. Ouvrir une session terminée et lancer le contrôle de clôture.
5. Vérifier que les émargements d’une session clôturée sont en lecture seule.
6. Ouvrir Documents et vérifier le classement par session et catégorie.

Ne pas redéployer `process-email-queue`. Ne pas modifier NCR Academy.
