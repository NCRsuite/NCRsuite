# NCR Suite V2.12.1 — Suppression sécurisée des entreprises

## Installation

1. Exécuter `supabase/migrations/062_platform_organization_secure_deletion.sql`.
2. Déployer l’Edge Function `supabase/functions/admin-delete-organization/index.ts` sous le nom `admin-delete-organization` avec **Verify JWT activé**.
3. Ajouter les autres fichiers du patch dans GitHub en conservant leur arborescence.
4. Attendre le déploiement Cloudflare, puis fermer et rouvrir la PWA.

## Utilisation

Dans **Administration NCR → Entreprises**, sélectionner une entreprise. La zone de danger permet au super-administrateur de supprimer définitivement l’entreprise, même active. Le nom exact doit être saisi avant validation.

La suppression efface les données rattachées à l’entreprise et les fichiers dont le chemin commence par son identifiant dans les buckets NCR Suite connus. Les comptes Auth ne sont pas supprimés.
