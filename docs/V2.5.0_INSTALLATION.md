# NCR Suite V2.5.0 — Installation

Cette version ouvre le domaine **Sécurité privée** avec le socle de l’offre Découverte.

## Ordre obligatoire

1. Dans **Supabase → SQL Editor**, exécuter entièrement :

   `supabase/migrations/027_security_discovery_core.sql`

2. Attendre le message **Success**.
3. Envoyer ensuite tous les fichiers du patch sur GitHub en conservant l’arborescence.
4. Attendre le build **Success** de Cloudflare.
5. Fermer puis rouvrir la PWA pour charger le cache V2.5.0.

## Fonctions installées

- ouverture des espaces Sécurité dans l’administration NCR ;
- clients donneurs d’ordre ;
- sites surveillés avec tarif horaire HT ;
- agents sans accès terrain pour l’offre Découverte ;
- planning hebdomadaire avec contrôle des chevauchements ;
- calcul des heures programmées ;
- préfacturation par client et par période ;
- PDF de préfacturation ;
- tableau de bord Sécurité réel.

La formule de calcul est strictement : **heures programmées × tarif horaire du site**.

Aucun redéploiement de Brevo ou de `process-email-queue` n’est nécessaire.
