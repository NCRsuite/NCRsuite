# NCR Suite V2.8.4 — Paramètres Restauration et page publique

## Correctifs

- Le tarif affiché dans les paramètres correspond désormais au métier Restauration.
- Les droits de la formule Restauration sont présentés avec les bonnes fonctions.
- La page publique de réservation peut être activée depuis Paramètres ou directement depuis Réservations.
- Le lien utilisé est `/r/{slug}/reserver`.
- Le droit de réservation en ligne est réparé pour Essentielle, Professionnelle et Métier.

## Installation

1. Exécuter `supabase/migrations/049_restaurant_public_booking_settings.sql` dans Supabase.
2. Ajouter les autres fichiers du patch dans GitHub en conservant leur arborescence.
3. Vérifier le déploiement Cloudflare.
4. Fermer puis rouvrir la PWA pour charger `ncr-suite-shell-v2.8.4-restaurant-public-page`.

## Test

1. Ouvrir une entreprise Restauration en offre Professionnelle.
2. Aller dans Paramètres et vérifier `79,90 € HT / mois · tarif Restauration`.
3. Activer la réservation publique et enregistrer.
4. Ouvrir la rubrique Réservations puis la page publique.
5. Vérifier que l’URL utilise `/r/<slug>/reserver` et que le formulaire public s’affiche.
