# NCR Suite V2.4.8 — Mise à jour depuis la V2.4.7

## Ordre d’installation

### 1. Supabase

Aucun SQL à lancer pour cette version.

### 2. GitHub

Décompresser le patch à la racine du dépôt NCR Suite V2.4.7, puis remplacer ou ajouter uniquement les fichiers fournis.

Le nouveau fichier à ajouter est :

- `src/lib/browserFiles.ts`

Les autres fichiers du patch remplacent leurs versions V2.4.7.

### 3. Cloudflare

Valider le commit GitHub et attendre la fin du déploiement Cloudflare.

Le build doit exécuter normalement `npm clean-install` puis `npm run build`.

### 4. Cache PWA

Après le déploiement, fermer complètement NCR Suite puis rouvrir l’application afin de charger le cache V2.4.8.

### 5. Tests à réaliser

1. Ouvrir **Formation → Attestations**.
2. Tester **Visualiser** sur une attestation existante.
3. Tester **Télécharger** sur la même attestation.
4. Ouvrir une session contenant des signatures matin et après-midi.
5. Générer puis visualiser la feuille d’émargement PDF.
6. Vérifier que les signatures apparaissent dans les bonnes cellules.
7. Tester le bouton **Télécharger** du PDF d’émargement sur iPhone ou Mac.

## Éléments non modifiés

- aucune table ni règle Supabase ;
- aucune Edge Function ;
- aucune automatisation Brevo ;
- aucun module Coiffure ;
- aucun fichier NCR Academy.
