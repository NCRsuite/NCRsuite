# NCR Suite V2.8.6 — Expérience Restauration Premium

## Installation

Cette version est un correctif visuel et ergonomique appliqué par-dessus la V2.8.5.

1. Aucun SQL Supabase n’est nécessaire.
2. Remplacer ou ajouter les fichiers du patch dans GitHub en conservant leur arborescence.
3. Vérifier le déploiement Cloudflare.
4. Fermer complètement la PWA puis la rouvrir.

Cache PWA : `ncr-suite-shell-v2.8.6-restaurant-premium`.

## Écrans concernés

- Prise de commande tactile.
- Note provisoire.
- Carte et gestion des plats.
- Menu QR public multilingue.
- Écran cuisine et tickets de préparation.

La base de données, les commandes existantes, le plan de salle, les réservations et les moyens de paiement externes ne sont pas modifiés.
