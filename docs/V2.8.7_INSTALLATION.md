# NCR Suite V2.8.7 — Traduction complète du menu public

## Installation

1. Aucun SQL Supabase n’est nécessaire.
2. Remplacer les fichiers du patch dans GitHub en conservant l’arborescence.
3. Vérifier le déploiement Cloudflare.
4. Fermer puis rouvrir la PWA afin de charger le cache `ncr-suite-shell-v2.8.7-public-menu-i18n`.

## Contrôle

Ouvrir le menu public et sélectionner FR, EN, ES puis IT. Le bandeau d’accueil, les compteurs, les titres de section, les badges, les messages d’état et le pied de page doivent tous changer de langue.
