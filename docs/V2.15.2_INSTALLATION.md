# NCR Suite V2.15.2 — Formation · Déroulement et clôture automatisés

## Installation

1. Exécuter `supabase/migrations/073_training_delivery_closure_automation.sql` dans Supabase.
2. Copier les fichiers du patch sur GitHub en conservant l’arborescence.
3. Redéployer l’Edge Function `process-email-queue`.
4. Vérifier le build Cloudflare en version `2.15.2`.
5. Fermer et rouvrir la PWA pour renouveler le cache.

## Parcours livré

- La validation d’une session déclenche les convocations et les évaluations initiales.
- Les questionnaires sont individuels, publics et reliés au dossier de session.
- Les relances Brevo sont générées automatiquement par le processeur d’e-mails.
- La fin de session déclenche les évaluations finales.
- Selon les réglages, l’attestation est générée immédiatement ou après la réponse finale.
- Le dossier est finalisé automatiquement lorsque les éléments obligatoires sont réunis.

## Réglages

La page **Formation → Évaluations** permet de régler une seule fois :

- activation et délai de l’évaluation initiale ;
- activation et délai de l’évaluation finale ;
- délai et nombre de relances ;
- envoi automatique de l’attestation ;
- obligation de répondre à l’évaluation finale avant l’attestation.
