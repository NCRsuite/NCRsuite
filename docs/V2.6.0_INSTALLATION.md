# Installation NCR Suite V2.6.0

## 1. Migration Supabase

Exécuter `supabase/migrations/037_global_push_notifications.sql` dans le SQL Editor et attendre `Success`.

La migration crée :

- les préférences par entreprise et utilisateur ;
- les abonnements Web Push par appareil ;
- le centre de notifications ;
- la file d’envoi ;
- les déclencheurs Coiffure, Formation et Sécurité ;
- la planification `pg_cron` utilisée après initialisation.

## 2. Edge Functions

Déployer via **Supabase → Edge Functions → Deploy a new function → Via Editor** :

1. `setup-push-notifications` avec le contenu de `supabase/functions/setup-push-notifications/index.ts` ;
2. `process-push-notifications` avec le contenu de `supabase/functions/process-push-notifications/index.ts`.

Aucun secret supplémentaire n’est à créer manuellement. L’initialisation depuis l’administration NCR génère les clés VAPID et le secret interne du worker.

Ne pas modifier `process-email-queue` ni `send-security-document`.

## 3. GitHub et Cloudflare

Déposer le patch à la racine de la V2.5.9, conserver l’arborescence, valider le commit et attendre `Success` dans Cloudflare.

## 4. Initialisation centrale

Se connecter avec le compte super-administrateur NCR, ouvrir **Notifications push**, puis cliquer sur **Initialiser les notifications push**.

Cette action :

- génère et conserve les clés VAPID ;
- configure l’appel automatique de la fonction d’envoi chaque minute ;
- n’expose jamais la clé privée aux entreprises ou aux navigateurs.

## 5. Activation sur les appareils

Chaque utilisateur ouvre **Notifications** dans son espace, choisit ses préférences, puis clique sur **Activer les notifications**.

Sur iPhone/iPad, les notifications Web Push nécessitent l’ouverture de NCR Suite depuis l’icône ajoutée à l’écran d’accueil. Le téléphone peut ensuite recevoir les alertes lorsqu’il est verrouillé.

## Premiers événements automatiques

- Coiffure : nouveau rendez-vous, modification, annulation et rappel ;
- Formation : session créée/modifiée, rappel 24 h avant et document publié ;
- Sécurité : vacation créée/modifiée, rappel 1 h avant, alerte de site, PTI/SOS et statuts de facture ;
- Tous les domaines : centre de notifications et préférences individuelles.
