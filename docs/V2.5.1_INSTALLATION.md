# NCR Suite V2.5.1 — Installation

Cette version ajoute les fonctions de l’offre **Sécurité Essentielle** au socle V2.5.0.

## Ordre obligatoire

1. Dans **Supabase → SQL Editor**, exécuter entièrement :

   `supabase/migrations/028_security_essential_field.sql`

2. Attendre le message **Success**.
3. Envoyer ensuite tous les fichiers du patch sur GitHub en conservant l’arborescence.
4. Attendre le build **Success** de Cloudflare.
5. Fermer puis rouvrir la PWA pour charger le cache V2.5.1.

## Contrôles recommandés

1. Passer une entreprise Sécurité sur l’offre Essentielle.
2. Créer un agent et lui envoyer une invitation depuis **Accès agents**.
3. Accepter l’invitation avec l’adresse concernée.
4. Vérifier que l’agent ne voit que son espace terrain, son planning, les rondes, la main courante et les consignes.
5. Créer un point de ronde, télécharger son QR puis tester sa lecture depuis un téléphone. Le décodage est effectué localement sur l’appareil ; utiliser le code manuel de secours si la photo est illisible.
6. Ajouter une entrée de main courante depuis un choix prédéfini.
7. Publier une consigne et une alerte pour un site, puis confirmer sa lecture côté agent.
8. Dupliquer une mission sur plusieurs jours de la semaine.
9. Générer le planning mensuel PDF d’un agent et vérifier les couleurs des sites, les horaires et les totaux.
10. Générer le PDF de main courante côté responsable.

## Limites conservées

- Offre Découverte : aucun compte agent terrain.
- Offre Essentielle : 10 agents connectés maximum.
- Géolocalisation, PTI/SOS, supervision temps réel et rôle Chef de poste restent réservés à la future offre Professionnelle.

Aucun redéploiement de Brevo ou de `process-email-queue` n’est nécessaire. NCR Academy n’est pas concernée.
