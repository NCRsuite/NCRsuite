# NCR Suite V2.6.6 — Centre opérationnel Sécurité

## Contenu
- Nouveau centre opérationnel QG sur le tableau de bord Sécurité.
- Compteurs en temps réel : agents en poste, prises de poste en retard, fins de poste oubliées et dossiers à clôturer.
- Liste des agents actuellement en poste avec accès direct au dossier de vacation.
- File de priorités QG regroupant alertes critiques, retards et fins de poste non enregistrées.
- Mise en page responsive mobile et bureau.

## Installation
1. Remplacer les fichiers du dépôt par ceux de cette version.
2. Aucun nouveau SQL Supabase n'est requis pour ce lot.
3. Déployer sur Cloudflare Pages.
4. Vider le cache de la PWA ou fermer puis rouvrir l'application.
5. Vérifier le tableau de bord avec un compte QG Sécurité.

## Tests recommandés
- Une vacation commencée apparaît dans « agents en poste ».
- Une vacation non commencée depuis plus de 15 minutes apparaît dans « prises en retard ».
- Une vacation dépassée sans fin de poste apparaît dans les priorités.
- Les liens ouvrent la rubrique Dossiers vacations.
- Les comptes Agent conservent leur tableau de bord terrain actuel.
