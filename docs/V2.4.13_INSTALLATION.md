# Installation NCR Suite V2.4.13

## 1. Supabase d’abord

Exécuter entièrement dans le SQL Editor :

```text
supabase/migrations/026_central_offer_catalog.sql
```

Attendre le message `Success` avant de poursuivre.

La migration :

- crée le catalogue des offres Sécurité, Nettoyage et Restauration ;
- remplace l’ancien domaine Artisan par Restauration ;
- conserve Coiffure et Formation sans modifier leurs fonctions ;
- bloque la création des domaines encore en préparation ;
- prépare les modules Restauration pour une future version.

## 2. GitHub ensuite

Déposer tout le contenu du patch à la racine de la V2.4.12 en conservant les dossiers, puis accepter les remplacements.

## 3. Cloudflare

Attendre le statut `Success`.

## 4. Cache PWA

Fermer complètement NCR Suite puis rouvrir l’application.

## 5. Contrôles

- ouvrir `Administration NCR → Catalogue des offres` ;
- vérifier les cinq domaines et leurs tarifs ;
- vérifier que Sécurité, Nettoyage et Restauration sont indiqués `En préparation` ;
- ouvrir une entreprise Coiffure puis Formation et contrôler que leurs abonnements restent inchangés ;
- vérifier que l’écran de création d’espace ne permet encore que Coiffure et Formation.
