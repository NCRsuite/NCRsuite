# NCR Suite V2.5.3 — Installation

1. Exécuter `supabase/migrations/030_security_qr_billing_fix.sql` dans Supabase SQL Editor.
2. Attendre le message `Success`.
3. Envoyer ensuite le contenu du patch à la racine du dépôt GitHub V2.5.2.
4. Attendre le build Cloudflare `Success`.
5. Fermer puis rouvrir la PWA pour charger le cache V2.5.3.
6. Tester une ronde QR avec la caméra, puis une préfacture Sécurité.
