# Installation — NCR Suite V2.15.1

## Pré-requis

La V2.15.0 doit être installée et validée.

## 1. Migration Supabase

Exécuter :

```text
supabase/migrations/071_training_premium_documents_brevo.sql
```

La migration ajoute le suivi des PDF commerciaux, leur mise en file Brevo, le modèle d’e-mail commercial et l’enregistrement de la signature et du cachet documentaire.

## 2. Fichiers GitHub

Copier tous les fichiers du patch en conservant les chemins.

## 3. Edge Function

Redéployer obligatoirement :

```text
supabase/functions/process-email-queue/index.ts
```

Cette fonction génère les convocations et attestations premium, prépare l’e-mail commercial et attache le PDF stocké dans le bucket privé `training-documents`.

## 4. Cloudflare et PWA

Le build doit afficher `ncr-suite@2.15.1`.

Cache attendu :

```text
ncr-suite-shell-v2.15.1-training-documents
```

Fermer ensuite complètement la PWA et la rouvrir.

## 5. Contrôles

- Programme PDF depuis une fiche formation complète.
- Devis, convention et contrat premium.
- Envoi Brevo et pièce jointe.
- Adresse Reply-To correspondant au Profil organisme.
- Convocation sans chevauchement entre le logo et l’adresse.
- Signature et cachet visibles lorsqu’ils sont configurés.
