# Installation NCR Suite V2.5.5

## 1. Supabase

Exécuter dans le SQL Editor :

```text
supabase/migrations/032_security_branding_final_invoices.sql
```

Attendre le message `Success`.

## 2. GitHub

Déposer ensuite tout le contenu du patch à la racine de la V2.5.4 en conservant l’arborescence.

## 3. Cloudflare

Attendre la fin du build et le statut `Success`.

## 4. Cache

Fermer complètement la PWA puis la rouvrir.

## 5. Contrôles

1. Dans **Sécurité > Personnalisation**, ajouter le logo et compléter le profil de facturation.
2. Générer une main courante et un planning PDF : le logo doit apparaître.
3. Dans le planning, marquer une mission terminée comme **Réalisée**.
4. Dans **Facturation > Factures définitives**, valider la durée réellement effectuée.
5. Émettre puis télécharger la facture.
6. Marquer la facture envoyée puis payée.
7. Vérifier que les vacations déjà facturées ne réapparaissent plus dans le calcul.

## Logique retenue

- **Préfacture** : heures programmées × tarif du site.
- **Facture définitive** : minutes réalisées et validées × tarif du site.
- La facture reçoit un numéro chronologique `FAC-AAAA-000001`.
- Les coordonnées de l’émetteur et du client sont figées lors de l’émission.
- Une vacation ne peut être liée qu’à une seule facture définitive.

Cette version ne met pas en place les avoirs ni une connexion à une plateforme de facturation électronique.
