NCR SUITE V2.12.3 — PATCH UNIQUEMENT
========================================

BASE OBLIGATOIRE
- Installer ce patch uniquement sur la V2.12.2 validée.
- Ne pas supprimer les fichiers absents du ZIP.
- NCR Academy est un projet séparé et ne doit pas être modifié.

ORDRE D’INSTALLATION
1. Supabase > SQL Editor
   Exécuter intégralement :
   supabase/migrations/064_coiffure_loyalty_client_portal.sql

2. Supabase > Edge Functions
   Redéployer la fonction existante :
   supabase/functions/process-email-queue/index.ts

3. GitHub
   Copier tous les fichiers du patch en conservant exactement leurs chemins.
   Remplacer les fichiers existants lorsque GitHub le demande.

4. Cloudflare
   Vérifier que le build se termine sans erreur et que la version affichée est 2.12.3.

5. PWA
   Fermer complètement l’application puis la rouvrir afin de charger le cache :
   ncr-suite-shell-v2.12.3-coiffure-loyalty-portal

TESTS À EFFECTUER
- Ouvrir Coiffure > Fidélité.
- Activer uniquement les règles souhaitées par le salon.
- Terminer un rendez-vous test et vérifier le crédit de points/passages.
- Envoyer une invitation à un client et activer son espace.
- Vérifier les rendez-vous, les avantages et les préférences anniversaire.
- Marquer une récompense comme utilisée.
- Suspendre puis réactiver l’accès client.

IMPORTANT
- Les rendez-vous déjà terminés avant l’installation ne sont pas crédités rétroactivement.
- Leur solde initial peut être repris avec l’outil « Ajustement manuel ».
